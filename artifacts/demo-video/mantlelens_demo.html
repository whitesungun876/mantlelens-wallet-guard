Problem

Agents need evidence before they act.

Wallet tools often show risk as a final score. MantleLens shows the signal, the evidence, the unknown fields, and the action boundary.

The dangerous shortcut

"No data found" is not the same as "no risk found." Missing indexed data may hide approvals, unknown tokens, or transfer history.

MantleLens answer

Separate risk score, detected signals, data coverage, proof status, record status, and outcome. Each page tells one consistent story.

The agent treats incomplete coverage as unknown, not safe.

Agent decision audit

Rules and evidence run before explanation.

MantleLens exposes the decision path so judges can see it is not a free-form LLM wrapper.

Decision Review transfer evidence
Action Check source coverage
Simulation Unavailable

Why this decision

  • Transfer evidence requires review.
  • Additional source coverage gaps exist.
  • Unknown coverage cannot be marked safe.
  • No approval or yield action is available to simulate.

Safety boundary

  • LLM explains; it does not execute.
  • Real revoke is blocked in the default demo path.
  • No swap, transfer, auto-signing, or private-key custody.
  • Assessment proof records the assessment hash, not wallet safety.
The decision is evidence-bound and non-custodial.
